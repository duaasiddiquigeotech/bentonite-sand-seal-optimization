# Optimisation of Bentonite–Sand Mixtures for Gas-Permeable Seals

## Overview

This repository presents my MSc thesis research on the design and optimisation of bentonite–sand mixtures for use as gas-permeable seals in deep geological radioactive waste disposal.

The engineering challenge is to design a material that:

- maintains very low hydraulic conductivity,
- develops sufficient swelling pressure,
- allows controlled gas migration,
- and reaches saturation within an acceptable time.

## Research Objective

The objective was to identify combinations of bentonite content and bentonite dry density that satisfy several competing engineering requirements simultaneously.

## Methodology

The project combines:

- coupled hydro-mechanical finite element modelling,
- constitutive modelling of unsaturated soils,
- experimental-data validation,
- empirical correlation development,
- parametric analysis,
- sensitivity analysis,
- and engineering design optimisation.

## Main Performance Indicators

The following material properties were evaluated:

- swelling pressure,
- saturated hydraulic conductivity,
- gas-entry pressure,
- water-retention behaviour,
- and saturation time.

## Numerical Tools

- LAGAMINE
- Python
- Finite Element Method
- Barcelona Basic Model
- Parametric and sensitivity analysis

## Key Outcome

The analysis identified a feasible design region in which the main hydraulic, mechanical, and gas-transport requirements were satisfied simultaneously.
